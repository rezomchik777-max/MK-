# Pomodoro Kombat

Полный Android-проект с GitHub Actions.

Фичи: русский/английский интерфейс, режимы 25/15/50/кастом, ретро-звуки через системный ToneGenerator, вибрация, уведомления, XP, победы, серии, темы Scorpion/Sub-Zero, AMOLED-стиль.

## Сборка через GitHub
1. Залей содержимое этой папки в репозиторий.
2. Открой Actions → Build Android APK → Run workflow.
3. Скачай Artifacts → PomodoroKombat-debug-apk.
4. Внутри будет app-debug.apk.
